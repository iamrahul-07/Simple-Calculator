# Simple-Calculator


<h1>Overview<h1>
<p>
This calculator is a user-friendly graphical user interface built using the Tkinter library in Python.
It is designed to provide a simple and efficient way to perform basic arithmetic operations, as well as additional functions
like square root, square, pi, and factorial. Whether you're a student, professional, or anyone in need of a versatile calculator, this application has you covered.
</p>
<h2>Features<h2>
<br>
<h3>Basic Arithmetic Operations:<h3>
<br>
<p>
1.Addition<br>
2.Subtraction<br>
3.Multiplication<br>
4.Division<p><br>
</p>
<h3>Additional Functions:<h3><br>
<p>
1.Square Root (√)<br>
2.Square (x²)<br>
3.Pi (π)<br>
4.Factorial (x!)<br>
</p>
<h3>Clear Function:<h3><br>
<p>Easily clear the current input to start a new calculation.</p><br>
<h3>Decimal Point Support:<h3><br>
<ul>
<p>Perform calculations with decimal numbers for increased precision.</p><br>
<ul>
<h3>User-friendly Interface:<h3>
<ul><br>
<p>Intuitive buttons for numbers, operations, and special functions.<br>
Responsive design for seamless interaction.</p><br>
<ul>
<h2>Usage<h2>
<br><br>
<p>The Tkinter Calculator provides a visually appealing and straightforward interface:</p><br>
<h3>Number Buttons:<h3><br>
<p>Click on the number buttons to input digits.<br>
0 - 9 are represented by their respective button labels.</p><br>
<h3>Operation Buttons:<h3><br>
<p>Click on the operation buttons (+, -, *, /) to choose the desired operation.</p><br>
<h3>Function Buttons:<h3><br>
<p>Click on the special function buttons (√, x², π, x!) to perform advanced calculations</p><br>
<h3>Equals Button:<h3><br>
<p>Click on "=" to perform the calculation and display the result.</p><br>
<h3>Clear Button:<h3><br>
<p>Click on "C" to clear the current input.</p><br>
<h3>Decimal Point Button:<h3><br>
<p>Click on "." to input decimal points for precision.</p><br>
